# Tactical C2 / ICS i18n Pack

Files:
- zh-TW.json: Traditional Chinese (Taiwan) UI strings
- en.json: English UI strings
- ui-copy.meta.json: per-key maxChars & suggested abbreviations (zh/en)

Placeholders:
- Uses {name} format, e.g. {count}, {time}, {level}

Intended use:
- i18next / react-intl / vue-i18n etc.
