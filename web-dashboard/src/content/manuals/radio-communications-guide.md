# 第 0 章：使用前必讀

> ⚠️ **本章為所有使用者必讀內容**

## 合法使用原則

### 無線電法規基本認知

台灣的無線電管理由 **國家通訊傳播委員會（NCC）** 主管。根據《電信管理法》及相關規定：

1. **無線電頻率是國家資源**，需依規定使用
2. **非法發射可處罰鍰或刑責**
3. **不同頻段有不同使用規定與執照要求**

### 執照概念

| 類型 | 說明 | 常見應用 |
|------|------|---------|
| **業餘無線電執照** | 需通過考試取得，分等級 | 登山、救災通訊、業餘愛好 |
| **專用電臺執照** | 機構申請，有頻率/功率限制 | 計程車、工廠、保全 |
| **船舶電臺執照** | 船舶使用海事頻段需申請 | 漁船、遊艇 |
| **免執照頻段** | 限定功率與頻段內可免執照使用 | 對講機（受規範限制） |

> [!WARNING]
> **發射行為需執照**：僅「收聽」通常不違法，但「發射」需符合執照/頻段規定。購買無線電設備不代表可以任意發射使用。

### 不得干擾原則

根據法規，使用無線電時：

1. **不得干擾合法通訊**
2. **不得使用未經核准的頻率**
3. **不得超過法定功率限制**
4. **需忍受其他合法電臺的干擾**

### 緊急情境注意事項

1. **生命危急時可使用任何頻率求救**（但事後需能說明情況）
2. **緊急呼叫不可濫用**，謊報求救可能觸法
3. **國際求救頻率**：海上 VHF 16 頻道、航空 121.5 MHz 為緊急頻率
4. **山難/迷途建議優先使用手機撥打 112 或 119**

> [!CAUTION]
> **本手冊僅供教育參考**。無線電法規可能變動，使用前請查閱 NCC 官網最新公告。違法使用可能面臨罰鍰新臺幣 3 萬至 100 萬元，或設備沒收。

# 第 1 章：無線電基本概念

## 名詞解釋

### 頻率 (Frequency)

- **定義**：電磁波振盪的速度，單位為赫茲 (Hz)
- **常見單位**：
  - kHz（千赫）= 1,000 Hz
  - MHz（兆赫）= 1,000,000 Hz
  - GHz（吉赫）= 1,000,000,000 Hz
- **類比**：如同廣播電台的「台號」，不同頻率用於不同通訊

### 頻道 (Channel)

- **定義**：預設的頻率組合，便於快速切換
- **說明**：機器上的「1 頻道」「2 頻道」是設定好的頻率，不同品牌/設定可能對應不同頻率

### 頻段 (Band)

- **定義**：一段連續的頻率範圍
- **常見業餘頻段**：
  - VHF（Very High Frequency）：144-146 MHz
  - UHF（Ultra High Frequency）：430-440 MHz

### 調變方式

| 類型 | 說明 | 特性 |
|------|------|------|
| **FM**（調頻）| 聲音控制電波頻率變化 | 抗干擾佳、音質清晰、消耗較多頻寬 |
| **AM**（調幅）| 聲音控制電波振幅變化 | 傳輸遠、易受干擾、航空通訊使用 |

### 功率 (Power)

- **定義**：發射電波的強度，單位為瓦特 (W)
- **與距離的關係**：功率越大通訊距離越遠，但耗電也越大
- **法規限制**：不同執照/頻段有功率上限
- **常見設定**：
  - 低功率（L）：約 0.5-1W，省電、近距離
  - 高功率（H）：約 5W，遠距離但耗電

### 天線

- **功能**：將電子訊號轉為電波發送，或接收電波轉為訊號
- **影響**：天線品質與匹配度直接影響通訊距離與品質
- **基本原則**：天線應垂直持握，避免遮擋或接觸身體

### SWR（駐波比）

- **定義**：衡量天線與發射機匹配程度的數值
- **理想值**：1:1（完美匹配）
- **可接受值**：1.5:1 以下
- **過高的風險**：能量反射回發射機，可能造成損壞

### 同頻干擾

- **現象**：多人同時使用相同頻率會互相干擾
- **解決方式**：
  - 使用 CTCSS/DCS 編碼（後述）
  - 協調使用不同頻率
  - 遵守通聯禮儀，避免長時間佔用

# 第 2 章：無線電種類與選購

## 設備類型比較

| 類型 | 功率 | 特性 | 適用場景 |
|------|------|------|---------|
| **手持機** | 0.5-8W | 便攜、電池供電 | 登山、露營、活動 |
| **車機** | 10-50W | 車輛供電、功率大 | 車隊通訊、基地補給 |
| **基地台** | 10-100W+ | 固定位置、需電源 | 指揮中心、中繼站 |
| **中繼台** | 依設計 | 擴大通訊範圍 | 跨山頭、偏遠地區 |

## 類比 vs 數位

| 特性 | 類比 | 數位（如 DMR） |
|------|------|---------------|
| 音質 | 隨距離衰減 | 清晰或斷訊（無中間態） |
| 抗干擾 | 較差 | 較佳 |
| 設備成本 | 較低 | 較高 |
| 通訊容量 | 一頻一話 | 可多工 |
| 相容性 | 廣泛 | 需同系統 |

## 常見規格說明

| 規格 | 說明 | 選購建議 |
|------|------|---------|
| 頻段 | VHF / UHF / 雙頻 | 雙頻較彈性 |
| 功率 | 瓦特數 | 手持 5W 足夠一般戶外 |
| 頻道數 | 可存幾組頻率 | 建議 16 頻道以上 |
| 防水等級 | IP 等級 | 戶外建議 IP54 以上 |
| 電池容量 | mAh | 建議 1500mAh 以上 |

## 選購建議

### 一般戶外活動（登山、露營）
- **建議**：VHF/UHF 雙頻手持機
- **功率**：5W
- **功能**：CTCSS/DCS、防水、備用電池

### 車隊/編隊活動
- **建議**：車機 + 手持機組合
- **功率**：車機 25-50W、手持 5W
- **功能**：車充、外接天線

### 救災/志工應用
- **建議**：與團隊相同品牌/規格
- **功能**：掃描、雙守、長效電池

> [!IMPORTANT]
> **購買不等於可用**：部分無線電設備可發射的頻率需執照才能使用。購買前應確認用途與法規限制。

# 第 3 章：操作入門 SOP

## 基本操作流程

### 步驟 1：開機與檢查

1. **長按電源鍵**開機
2. **確認電量**：電池符號、電量百分比
3. **確認頻率/頻道**：螢幕顯示當前設定
4. **確認音量**：轉至適當大小

### 步驟 2：設定頻率/頻道

**使用預設頻道**：
1. 按「頻道選擇鍵」或轉動旋鈕
2. 選擇團隊協議的頻道編號

**手動輸入頻率**（如機器支援）：
1. 切換至 VFO 模式（自由頻率模式）
2. 使用數字鍵輸入頻率（如 145.000）
3. 確認後存入頻道

### 步驟 3：收聽與掃描

**靜默收聽**：
1. 將音量調高
2. 等待其他人的呼叫
3. 確認頻道有人使用後再發話

**頻道掃描**：
1. 按「SCAN」鍵啟動掃描
2. 機器會自動巡迴各頻道
3. 收到訊號時暫停
4. 按 PTT 或特定鍵停止掃描

### 步驟 4：呼叫

1. **確認頻道無人使用**（聽 2-3 秒）
2. **按住 PTT（Push To Talk）**
3. **清晰說出呼叫內容**（詳見第 4 章）
4. **放開 PTT 等待回應**

### 步驟 5：通聯進行中

1. **按 PTT 說話，放開 PTT 聽對方**
2. **一次通話控制在 30 秒內**
3. **使用標準用語確認與復誦**
4. **保持簡短清晰**

### 步驟 6：結束通聯

1. **說「完畢」或「Over and out」**
2. **確認對方已收到**
3. **將機器設回待機狀態**

## 進階功能

### CTCSS / DCS 編碼

- **功能**：過濾不需要的通話，只接收同編碼的訊號
- **說明**：CTCSS（音頻編碼）和 DCS（數位編碼）可設定「次音頻」
- **用途**：在共用頻率時，讓團隊只聽到自己人的聲音
- **注意**：這不是加密，他人仍可聽到你

**設定方式（因機型而異）**：
1. 進入頻道設定選單
2. 找到 CTCSS 或 DCS 選項
3. 設定相同的編碼（如 88.5 Hz）
4. 收發雙方需設定一致

### 功率切換

- **低功率（L）**：省電、近距離使用
- **高功率（H）**：遠距離或訊號不佳時
- **建議**：平時用低功率，通訊不良時再切換

### 鎖鍵功能

- **功能**：防止誤觸改變設定
- **操作**：通常長按特定鍵啟動/解除

# 第 4 章：通聯禮儀與標準用語

## 基本禮儀原則

1. **先聽後說**：發話前確認頻道無人使用
2. **簡短清晰**：一次通話不超過 30 秒
3. **使用呼號**：說明自己是誰、找誰
4. **確認收到**：對方需回應「收到」
5. **禮貌結束**：說「完畢」後換對方發話

## 標準呼叫格式

### 發起呼叫

```
[對方呼號]、[對方呼號]，這裡是 [自己呼號]，聽到請回答。
```

**範例**：
```
「登山隊二號、登山隊二號，這裡是登山隊一號，聽到請回答。」
```

### 回應呼叫

```
[對方呼號]，[自己呼號] 收到，請說。
```

**範例**：
```
「登山隊一號，登山隊二號收到，請說。」
```

### 傳達訊息

```
[內容]，完畢。
```

**範例**：
```
「我們已到達三叉路口，在此等候，完畢。」
```

### 確認與復誦

```
收到，復誦：[重複對方內容]，是否正確？
```

**範例**：
```
「收到，復誦：你們在三叉路口等候，是否正確？」
```

### 結束通聯

```
收到，謝謝，[自己呼號] 完畢。
```

## 常用標準用語

| 用語 | 意義 |
|------|------|
| **收到** | 已聽到並理解 |
| **請說** | 請對方發話 |
| **完畢** | 我說完了，換你說 |
| **等一下** | 請稍候，我需要時間 |
| **復誦** | 重複確認內容 |
| **更正** | 我說錯了，以下是正確的 |
| **未收到** | 沒有聽清楚，請重說 |
| **訊號不佳** | 收訊品質差 |
| **請改頻道** | 請換到另一個頻道 |
| **緊急呼叫** | 有緊急狀況（慎用） |

## 通話紀錄

建議記錄：
- 時間
- 呼號
- 頻道
- 通話內容摘要
- 位置資訊

# 第 5 章：常見故障排除

## 故障診斷流程

```
                    ┌─ 完全沒聲音？
                    │   ├─ 電源是否開啟？
                    │   ├─ 音量是否太低？
                    │   └─ 電池是否沒電？
                    │
聽不到對方─────────┼─ 有雜音但沒聲音？
                    │   ├─ 頻率是否正確？
                    │   ├─ CTCSS/DCS 是否相同？
                    │   └─ 對方是否在發話？
                    │
                    └─ 聽到但很小聲？
                        ├─ 調高音量
                        └─ 距離太遠／有遮蔽

```

## 常見問題與解決

| 問題 | 可能原因 | 解決方法 |
|------|---------|---------|
| 完全聽不到 | 電源/音量/電池 | 檢查開機、調高音量、換電池 |
| 有雜音沒聲音 | 頻率錯誤 | 確認頻率與對方一致 |
| 聽不到特定人 | CTCSS/DCS 不符 | 確認編碼設定一致 |
| 對方聽不到我 | PTT 沒按好 | 確實按住 PTT 再說話 |
| 通話距離變短 | 電量低/天線問題 | 充電、檢查天線接觸 |
| 很多雜音 | 干擾/地形遮蔽 | 移動位置、提高功率 |
| 斷斷續續 | 邊界訊號/干擾 | 移動位置、確認頻道 |
| 機器當機 | 操作問題 | 關機重開 |

## 天線問題檢查

1. **天線是否鎖緊**：鬆動會嚴重影響訊號
2. **天線是否損壞**：彎折、斷裂需更換
3. **天線是否被遮擋**：手握位置不要蓋住天線
4. **天線方向**：垂直持握效果最佳

## 電池管理

1. **出發前充滿電**
2. **攜帶備用電池**
3. **平時使用低功率**
4. **不用時關機或省電模式**
5. **避免完全耗盡電量**

# 第 6 章：頻率/頻道指南

> [!CAUTION]
> **嚴正聲明**：本章僅列出「合法且公開可查證」的頻率資訊，供教育參考。任何發射行為請確認已取得相應執照。法規可能變動，請以 NCC 官方公告為準。

## 6.1 業餘無線電

### 概述

業餘無線電（Amateur Radio，俗稱「火腿族」）是經許可的無線電通訊活動，需通過考試取得執照。

### 執照等級

| 等級 | 考試難度 | 可用頻段/功率 |
|------|---------|--------------|
| 三等業餘 | 入門 | 限定頻段、功率較低 |
| 二等業餘 | 中階 | 更多頻段、功率提高 |
| 一等業餘 | 進階 | 全頻段、最大功率 |

### 常見業餘頻段（需執照）

| 頻段 | 頻率範圍 | 常見用途 | 來源 |
|------|---------|---------|------|
| 2m (VHF) | 144-146 MHz | 手持機通聯、中繼 | NCC 業餘無線電管理辦法 |
| 70cm (UHF) | 430-440 MHz | 手持機通聯、數位模式 | NCC 業餘無線電管理辦法 |

> [!WARNING]
> **發射需執照**：在上述頻段發射需持有業餘無線電執照。僅收聽不需執照，但仍需遵守相關規定。

### 中繼台概念

- **功能**：接收訊號後轉發，擴大通訊範圍
- **運作**：使用兩個頻率（一收一發），稱為「偏頻」
- **使用方式**：設定正確的收發頻率與 CTCSS
- **注意**：使用公開中繼台需遵守該台規定

### 查證方式

1. **NCC 官網**：https://www.ncc.gov.tw
2. **中華民國業餘無線電促進會（CTARL）**：http://www.ctarl.org.tw
3. **當地業餘無線電社團/俱樂部**

## 6.2 海事 VHF

### 概述

海事 VHF 用於船舶間及船岸通訊，有國際標準頻道配置。

### 公開頻道資訊

| 頻道 | 頻率 (MHz) | 用途 | 注意事項 | 來源 |
|------|-----------|------|---------|------|
| 16 | 156.800 | 國際遇險安全呼叫頻道 | 僅限緊急聯繫/呼叫，建立聯繫後需轉頻 | ITU 無線電規則 |
| 06 | 156.300 | 船間通訊 | 協調後使用 | ITU 無線電規則 |
| 09 | 156.450 | 船間通訊（備用呼叫） | 部分地區作為呼叫頻道 | ITU 無線電規則 |

> [!WARNING]
> **船舶電臺執照**：在船舶上使用海事 VHF 發射需有船舶電臺執照。16 頻道為緊急頻道，非緊急情況不得佔用。

### 監聽注意事項

1. 監聽海事頻道本身不違法
2. 聽到的資訊不得轉述或用於商業目的
3. 不得干擾船舶正常通訊

## 6.3 其他公開規範用途

### 免執照對講機（限定條件）

台灣對於低功率、特定頻段的設備有「免執照」規定，但仍需符合：
1. **設備需經 NCC 認證**（檢驗合格標籤）
2. **功率與頻率在規定範圍內**
3. **不得干擾其他合法通訊**

> [!IMPORTANT]
> **市售對講機不一定免執照**：許多市售「FRS」或「PMR」規格對講機在台灣可能不符合法規。購買前請確認是否有 NCC 認證標籤。

### 家用無線電話（DECT）

- 用於家庭內部的無線電話
- 特定頻段、低功率
- 設備需有 NCC 認證

## 6.4 如何查找與驗證頻率資訊

### 查證流程

```
1. 確認需求
   ↓
2. 查閱 NCC 官方公告及法規資料庫
   ↓
3. 查詢相關協會（CTARL、各地業餘俱樂部）
   ↓
4. 確認設備是否有 NCC 認證
   ↓
5. 確認個人是否有所需執照
   ↓
6. 如仍不確定，請勿發射
```

### 官方查證資源

| 資源 | 網址 | 用途 |
|------|------|------|
| NCC 官網 | https://www.ncc.gov.tw | 法規、公告、設備認證查詢 |
| 全國法規資料庫 | https://law.moj.gov.tw | 電信管理法、業餘無線電管理辦法 |
| CTARL | http://www.ctarl.org.tw | 業餘無線電資訊 |

### 設備認證確認

1. 查看設備是否有 **NCC 認證標籤**
2. 標籤上有認證號碼
3. 可至 NCC 網站查詢該號碼確認合法性

> [!CAUTION]
> **無法確認就不要用**：若無法確認某頻率是否合法或某設備是否認證，請勿發射使用。違法使用可能面臨處罰。

## 集合點與定時回報

### 集合點設定

1. 出發前確定**至少 2 個集合點**
2. 集合點需為**明顯地標**
3. 失散時往約定集合點移動
4. 每個集合點都要有**備選方案**

### 定時回報

| 回報間隔 | 建議情境 |
|---------|---------|
| 每 30 分鐘 | 危險地形、視線不佳 |
| 每 1 小時 | 一般行進 |
| 每 2 小時 | 輕鬆路線 |
| 到達重要點 | 休息站、叉路、山頂 |

**回報內容**：
1. 呼號
2. 目前位置
3. 隊伍狀況（人數、有無問題）
4. 下一個預計點

```

    ↓

    ↓

    ↓

    ↓

    ↓

    ↓
指揮判斷是否需要擴大搜索
    ↓
必要時撥打 119/112 報案
```

## 電量管理

### 省電策略

1. **平時低功率**：通訊順暢時使用低功率
2. **縮短通話**：說完就放開 PTT
3. **減少掃描**：掃描很耗電
4. **備用電池**：每人至少帶一組
5. **關機備用**：非值班者可關機

| 階段 | 建議電量 |
|------|---------|

| 返程時 | 30% 以上 |
| 緊急備用 | 保留一組滿電電池 |

## ❓ FAQ

**Q1：買了無線電機就可以直接使用嗎？**
> 不一定。需確認：(1) 設備是否有 NCC 認證 (2) 使用的頻率是否需要執照 (3) 功率是否在法定範圍。建議購買前先確認法規。

**Q2：業餘無線電執照怎麼考？**
> 至 NCC 網站查詢考試公告，報名參加筆試。三等業餘為入門級，通過後可申請發照。

**Q3：登山時用無線電會違法嗎？**
> 視情況。若使用業餘頻段需有業餘執照；若使用免執照頻段需確認設備符合規定。建議參加業餘無線電考試取得執照。

**Q4：別人在用的頻率我可以直接用嗎？**
> 同頻率會互相干擾。應先確認頻道無人使用再發話，或與團隊協調使用特定 CTCSS/DCS 編碼。

**Q5：緊急時可以用任何頻率求救嗎？**
> 生命危急時可使用任何手段求救，事後需能說明情況。但優先使用手機撥打 112/119 才是最有效方式。

**Q6：無線電可以聯繫多遠？**
> 取決於：功率、天線、地形、天氣。手持機（5W）在平地約 3-10 公里，山區因遮蔽可能僅數百公尺至數公里不等。

**Q7：CTCSS 和 DCS 是加密嗎？**
> 不是。它們只是「過濾」功能，讓相同編碼的電臺才會解除靜音。不同編碼的人仍可聽到你的發話。

**Q8：如何確認我的設備是合法的？**
> 檢查設備上是否有 NCC 認證標籤，並至 NCC 網站查詢認證號碼。無認證標籤者不可發射使用。
