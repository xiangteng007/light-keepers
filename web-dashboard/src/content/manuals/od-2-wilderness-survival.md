# 野外求生基本技能

> 迷途處置、保暖庇護、取水淨水、求援訊號完整指南

---

## 30 秒摘要

1. **迷路第一反應**：STOP 原則 — 停下來、冷靜思考、觀察環境、再做計畫
2. **保命優先順序**：遮蔽 → 保暖 → 水 → 食物（人可以 3 天不喝水、3 週不吃東西，但失溫可能 3 小時致命）
3. **求救訊號三連發**：聲音用三短聲、燈光用三閃、火用三堆
4. **待在原地**比亂走更容易被找到
5. **記住這句話**：「冷靜保命、原地等援、訊號持續」

---

## 適用情境 / 不適用情境

### ✅ 適用情境
- 登山健行迷路
- 露營時與隊伍走散
- 車輛故障受困山區
- 等待救援的臨時求生

### ❌ 不適用情境
- 海上求生（需專業海上求生訓練）
- 沙漠環境
- 極地生存
- 已有明確安全路線仍執意嘗試求生技能

## 風險優先原則

人體存活極限與處置優先順序：

| 優先順序 | 風險 | 極限時間 | 處置 |
|---------|------|---------|------|
| 1 | 窒息/溺水 | 3 分鐘 | 脫離危險、保持呼吸道暢通 |
| 2 | 失溫/中暑 | 3 小時 | 找遮蔽、保暖或降溫 |
| 3 | 脫水 | 3 天 | 尋找水源、節約體力 |
| 4 | 飢餓 | 3 週 | 保持體力、等待救援 |

> **口訣**：氣 → 溫 → 水 → 食

## 現場流程 SOP

### 步驟 1：STOP 原則（發現迷路時）

**目標**：冷靜下來，避免讓情況惡化

**S - Stop（停下來）**
- 發現迷路的當下，立刻停止移動
- 亂走只會離正確路徑更遠、消耗更多體力

**T - Think（思考）**
- 深呼吸，穩定情緒
- 回想最後一次確認位置是什麼時候、在哪裡

**O - Observe（觀察）**
- 觀察周圍地形：有無明顯地標、溪流、道路
- 檢查手機是否有訊號
- 評估天候狀況與剩餘日光

**P - Plan（計畫）**
- 決定：原地等待 or 嘗試返回（只有確定方向才能移動）
- 若不確定方向 → **待在原地是最佳選擇**

---

### 步驟 2：建立庇護所

**目標**：保護自己免受風雨、維持體溫

**選址原則**：
- 避風處（岩石背風面、濃密樹叢）
- 非低窪處（避免積水）
- 遠離落石區、枯木

**簡易庇護方式**（依可用資源）：

**方式 A：緊急毯帳篷**
1. 將緊急毯（鋁箔毯）拉開
2. 用繩索懸掛於兩樹間
3. 四角用石頭壓住
4. 鋁箔反光面朝外（可被搜救發現）

**方式 B：倒木庇護**
1. 找一根斜靠的倒木或大石
2. 在背風側用樹枝搭建斜面
3. 覆蓋樹葉、草堆增加隔熱

**方式 C：落葉床**
1. 清出一塊乾燥地面
2. 堆積 30cm 厚的落葉
3. 躺在落葉堆中，上方再覆蓋落葉

> **關鍵**：隔離地面寒氣比遮風更重要

---

### 步驟 3：保暖防失溫

**目標**：維持核心體溫

**保暖層次原則**：
1. **內層**：保持乾燥，濕衣服要換掉或擰乾
2. **中層**：製造空氣層（衣服不要太緊）
3. **外層**：擋風擋雨（緊急毯、塑膠袋皆可）

**沒有足夠衣物時**：
- 往衣服內塞乾落葉、乾草（製造空氣層）
- 將緊急毯裹在身上，反光面朝內
- 蜷曲身體減少散熱面積

**產熱方法**：
- 原地踏步、擺動手臂（避免劇烈運動耗盡體力）
- 生火取暖（下一步驟）
- 與同伴緊靠（體溫傳導）

---

### 步驟 4：生火

**目標**：取暖、烘乾衣物、煮沸水、製造求救訊號

**生火步驟**：
1. **選址**：避風處、遠離庇護所、清除周圍易燃物
2. **準備引火物**：
   - 最佳：樺樹皮、乾燥細草、衛生紙
   - 備選：口袋棉絮、繃帶碎片
3. **準備細柴**：拇指粗細的乾枝
4. **準備中柴**：手腕粗細的乾木
5. **堆疊結構**：引火物在底、細柴交叉堆疊、留空氣通道
6. **點火**：用打火機從底部點燃引火物
7. **逐步添加**：火勢穩定後再加中柴

**無打火機時**：
- 鏡子 + 陽光聚焦（需要耐心）
- 優先使用現有資源，不要耗盡體力

---

### 步驟 5：取水與淨水

**目標**：獲取安全飲用水

**尋找水源**：
- 順著山谷往下走通常可找到溪流
- 觀察動物足跡、蚊蟲聚集處
- 早晨收集葉面露水（用布擦拭後擰入容器）

**淨水方法（依優先順序）**：

| 方法 | 做法 | 有效度 |
|------|------|-------|
| 1. 煮沸 | 持續沸騰 3 分鐘 | ★★★★★ |
| 2. 淨水錠 | 依說明投入，等待 30 分鐘 | ★★★★☆ |
| 3. 濾水器 | 使用攜帶式濾水器過濾 | ★★★★☆ |
| 4. 紫外線曝曬 | 裝入透明瓶、曝曬 6 小時 | ★★★☆☆ |

**無法淨水時的選擇順序**：
1. 流動的清澈溪水（風險較低）
2. 雨水（較安全）
3. 靜止水源（風險高，非不得已不飲用）

> ⚠️ **絕對禁止**：飲用海水、尿液、酒精

---

### 步驟 6：發出求救訊號

**目標**：讓搜救人員發現你的位置

**國際求救訊號規則**：
- **三**代表求救（三聲、三閃、三堆火）

**聲音訊號**：
- 哨子：每隔 10 秒吹三短聲
- 沒有哨子：大喊「Help」或「救命」

**視覺訊號**：
- **日間**：鏡子對著太陽反射、揮動鮮豔衣物
- **夜間**：頭燈三閃、營火
- **空中搜救**：在空地排列 SOS 或 X（用石頭、樹枝）

**提高被發現機率**：
1. 待在空曠處（非密林中）
2. 將緊急毯攤開在地上（反光效果佳）
3. 聽到直升機立刻揮手並發出訊號
4. 夜間維持火堆

## 常見錯誤與後果

| 常見錯誤 | 後果 |
|---------|------|
| 1. 發現迷路繼續亂走「找路」 | 越走越遠，耗盡體力 |
| 2. 「等一下就會找到路」心態拖延 | 錯過日光，入夜後風險倍增 |
| 3. 沒告訴任何人就出發 | 失蹤後無人知道搜索區域 |
| 4. 飲用未處理的溪水 | 腹瀉脫水，惡化生存條件 |
| 5. 穿著濕衣服「走一走就乾」 | 失溫風險急劇上升 |
| 6. 認為有手機就不需要其他裝備 | 山區無訊號、電量耗盡 |
| 7. 使用全部體力嘗試求生技能 | 沒有體力撐到救援到來 |
| 8. 拋棄背包減輕重量 | 失去所有求生資源 |
| 9. 夜間趕路「想早點出去」 | 摔傷、迷失方向加劇 |
| 10. 食用不認識的植物 | 中毒風險極高 |

## 緊急狀況處置

### 🌀 完全迷失方向

**處置流程**：
1. 停止移動，執行 STOP 原則
2. 尋找最近的空曠高處
3. 確認手機是否有訊號（嘗試不同位置）
4. 若有訊號 → 撥打 112 告知情況
5. 若無訊號 → 建立庇護、發出訊號、等待救援

---

### 🥶 失溫

**症狀辨識**：
- 持續顫抖 → 手腳不協調 → 停止顫抖（危險！）→ 意識混亂 → 昏迷

**處置流程**：
1. 移至遮蔽處
2. 脫除濕衣、換乾衣或用緊急毯包裹
3. 製造暖源（火堆、體溫傳導）
4. 給予溫熱飲料（若意識清醒）
5. 中重度失溫 → 盡速求援

---

### 🦵 腳踝扭傷無法行走

**處置流程**：
1. 停止移動，坐下休息
2. RICE 原則：
   - R（Rest）休息
   - I（Ice）冷敷（用濕布或溪水降溫）
   - C（Compression）彈繃加壓
   - E（Elevation）抬高患肢
3. 評估是否能緩慢移動到較安全處
4. 無法移動 → 原地建立庇護、等待救援

---

### 🌧️ 突發暴雨

**處置流程**：
1. 立刻尋找遮蔽（岩石下、濃密樹叢）
2. 穿上雨衣或用緊急毯包裹
3. 遠離溪谷、低窪地
4. 若已搭庇護 → 確認排水、待在原地
5. 暴雨後檢查路況，確認安全再行動

## ❓ FAQ

**Q1：迷路時應該往上走還是往下走？**
> 沒有絕對答案。往下走可能找到溪流和道路，但也可能遇到斷崖。若不確定方向，**待在原地**是最安全選擇。

**Q2：沒帶食物可以撐多久？**
> 在有水的情況下，健康成人可存活 2-3 週。但會明顯虛弱，應減少活動、保存體力。

**Q3：可以吃野外的植物嗎？**
> 除非你 100% 確定辨識，否則不要吃。許多有毒植物外觀與可食植物相似，中毒會讓情況更糟。

**Q4：手機沒電了怎麼辦？**
> 使用非電子求救方法：哨子、鏡子反光、鮮豔衣物揮動、在地上排列 SOS。

**Q5：被困時可以喝自己的尿嗎？**
> 不建議。尿液含鹽分和廢物，會加速脫水。優先尋找其他水源。

**Q6：如何判斷是否該移動？**
> 只有在以下情況才移動：(1) 100% 確定正確方向 (2) 原地有立即危險 (3) 無人知道你的位置且確定不會有搜救。

**Q7：夜間迷路應該繼續走嗎？**
> 絕對不要。夜間行動風險極高（摔倒、迷失方向加劇）。找個安全處待到天亮。

**Q8：多久該發一次求救訊號？**
> 哨子：每隔 5-10 分鐘吹三聲。聽到任何聲音立刻持續回應。
