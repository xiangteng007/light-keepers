# 露營食安與飲水

> 食材保存、交叉污染預防、腹瀉處置、淨水流程完整指南

---

## 30 秒摘要

1. **冷鏈斷鏈 2 小時是關鍵**：肉類離開冰箱超過 2 小時即進入危險溫度帶
2. **生熟分離徹底執行**：不同砧板、不同器具、從生到熟單向操作
3. **水必經處理才安全**：煮沸 3 分鐘、淨水錠 30 分鐘、紫外線 6 小時
4. **腹瀉直接影響存活**：野外腹瀉會導致脫水，必須積極補水
5. **記住這句話**：「寧可餓、不可險；寧可渴、不亂喝」

---

## 適用情境 / 不適用情境

### ✅ 適用情境
- 露營區野炊
- 登山健行的食物準備
- 野外過夜的飲食管理
- 緊急避難時的食物處理

### ❌ 不適用情境
- 專業餐飲作業（需遵循更嚴格的法規）
- 有完善冷藏設備的長期野外基地
- 已確認受污染的食物（直接丟棄）

## 風險優先原則

| 優先順序 | 風險項目 | 處置原則 |
|---------|---------|---------|
| 1 | 飲用未處理生水 | 最高風險，必須淨化 |
| 2 | 食物中毒（細菌性） | 生熟分離、充分加熱 |
| 3 | 交叉污染 | 器具分開、流程單向 |
| 4 | 冷鏈斷裂 | 優先食用易腐食物、計算時間 |
| 5 | 食物過敏 | 事先確認成員過敏史 |

> **核心原則**：「不確定的不吃、來源不明的水不喝」

## 現場流程 SOP

### 步驟 1：食材保存管理

**目標**：延長安全食用期限、預防細菌滋生

**冷藏食材管理**：

| 時間 | 建議動作 |
|------|---------|
| 0-2 小時 | 保冷袋內，可安心使用 |
| 2-4 小時 | 優先使用，不可再冷藏 |
| 4 小時以上 | 建議丟棄 |

**保冷技巧**：
1. 冰磚放在食材**上方**（冷空氣往下沉）
2. 減少開啟次數
3. 放置於陰涼處，避免陽光直射
4. 保冷袋用鋁箔毯包覆增加隔熱

**無冷藏的食材選擇**：
- 第一天：可使用鮮肉、蛋類
- 第二天起：改用罐頭、乾燥食品、調理包

**⚠️ 必須丟棄的情況**：
- 肉類有異味或黏液
- 蛋殼裂開
- 罐頭膨脹或生鏽
- 食材在危險溫度（5-60°C）超過 4 小時

---

### 步驟 2：預防交叉污染

**目標**：阻斷病菌從生食傳到熟食

**生熟分離原則**：
1. **砧板**：生食用紅色標記、熟食用綠色標記
2. **刀具**：同上分開
3. **容器**：生食熟食用不同盆
4. **處理順序**：先處理熟食/蔬菜 → 最後處理生肉
5. **清洗**：處理生肉後，洗手並酒精消毒

**操作流程**：
```
【正確流程】
洗手 → 處理蔬菜 → 處理熟食 → 洗手換砧板 → 處理生肉 → 烹煮

【錯誤流程】❌
處理生肉 → 不洗手 → 處理蔬菜（污染！）
```

**手部衛生**：
- 接觸生肉、如廁後必須洗手
- 無水源時使用酒精消毒
- 戴手套處理生肉（使用後丟棄）

---

### 步驟 3：烹飪安全

**目標**：確保食物充分加熱、殺滅病菌

**安全溫度標準**：

| 食材 | 中心溫度 | 簡易判斷 |
|------|---------|---------|
| 家禽（雞、鴨） | ≥74°C | 肉質無粉紅、肉汁透明 |
| 絞肉/漢堡排 | ≥71°C | 無粉紅色 |
| 豬肉/牛肉 | ≥63°C | 可略粉紅（非絞肉） |
| 蛋類 | 蛋黃蛋白凝固 | 不流動 |
| 剩菜複熱 | ≥74°C | 完全冒熱氣 |

**野炊注意事項**：
1. 火力維持穩定，不要急著翻動
2. 肉類切小塊確保熟透
3. 湯品持續滾沸 3 分鐘以上
4. 不確定是否熟 → 繼續加熱

**⚠️ 禁止行為**：
- 嘗生肉測味道
- 熟食放回裝過生肉的盤子
- 剩菜放置一夜後未複熱就吃

---

### 步驟 4：飲水淨化

**目標**：獲取安全飲用水

**淨水方法比較**：

| 方法 | 步驟 | 時間 | 效果 |
|------|------|------|------|
| 煮沸 | 持續滾沸 | 3 分鐘 | ★★★★★ |
| 淨水錠 | 投入水中、等待 | 30 分鐘 | ★★★★☆ |
| 濾水器 | 過濾 | 即時 | ★★★★☆ |
| 紫外線曝曬 | 透明瓶裝水、日照 | 6 小時 | ★★★☆☆ |

**煮沸淨水 SOP**：
1. 取水時避開靜止水、污水源
2. 用布或紙巾過濾雜質
3. 放入鍋中加熱至滾沸
4. **持續沸騰 3 分鐘**（高山需 5 分鐘）
5. 放涼後裝入乾淨容器

**淨水錠使用 SOP**：
1. 依說明確認錠數與水量比例
2. 投入水中、輕搖混合
3. 等待 30 分鐘（部分產品需更久）
4. 水可能有輕微氯味，安全無虞

**⚠️ 不可飲用的水源**：
- 顏色異常（黃、綠、乳白）
- 有明顯臭味
- 靜止不流動的池塘
- 已知污染區域下游

---

### 步驟 5：剩菜處理

**目標**：避免細菌繁殖、預防隔餐食物中毒

**處理原則**：
1. 吃不完的熟食於 2 小時內保冰
2. 若無法保冰 → **當餐吃完或丟棄**
3. 隔餐食用前必須**完全複熱**（冒熱氣）
4. 複熱只能一次，不可反覆加熱

**野外建議**：
- 每餐只煮當餐份量
- 若不小心煮太多 → 趁熱分食完畢
- 寧願飯菜少一點，不要有剩菜風險

---

### 步驟 6：垃圾與廚餘處理

**目標**：不招引野生動物、維持環境

**處理原則**：
1. 廚餘密封後帶下山
2. 絕不掩埋（會被動物挖出）
3. 罐頭洗淨壓扁、帶回
4. 夜間將食物與垃圾吊高或遠離營地

## 常見錯誤與後果

| 常見錯誤 | 後果 |
|---------|------|
| 1. 把生肉和其他食材放同一袋 | 肉汁污染其他食材 |
| 2. 只帶一塊砧板「有洗就好」 | 洗不掉的細菌造成交叉污染 |
| 3. 看起來清澈的溪水直接喝 | 寄生蟲、細菌感染 |
| 4. 「肉看起來熟了」目測判斷 | 外熟內生，細菌存活 |
| 5. 剩菜放帳篷旁「明早再吃」 | 細菌繁殖超標、引來動物 |
| 6. 用裝過生蛋的碗裝熟食 | 沙門氏菌傳播 |
| 7. 洗食材的水直接拿來喝 | 生水中病菌量高 |
| 8. 手髒直接抓零食吃 | 病從口入 |
| 9. 保冷袋一整天沒看溫度 | 冰磚融化、冷鏈中斷不自知 |
| 10. 嫌帶回垃圾麻煩而掩埋 | 動物挖出、破壞環境 |

## 緊急狀況處置

### 🤢 食物中毒症狀

**常見症狀**：
- 噁心、嘔吐
- 腹痛、腹瀉
- 發燒
- 虛弱無力

**處置流程**：
1. **停止進食**可疑食物
2. **補充水分**（口服電解質液優先）
3. 記錄症狀發生時間與吃過的食物
4. 若症狀嚴重（血便、高燒、意識不清）→ **立即送醫**
5. 保留可疑食物樣本供檢驗

---

### 💧 野外腹瀉脫水

**脫水警訊**：
- 口渴、尿液減少且顏色深
- 皮膚彈性差（捏起後恢復慢）
- 頭暈、心跳加快

**處置流程**：
1. **持續小口補水**（不要大口灌，會引發嘔吐）
2. 每次腹瀉後補充 200ml 水 + 電解質
3. 暫時禁食固體食物，只喝稀釋果汁或電解質液
4. 若 24 小時內無改善或惡化 → 盡速撤離就醫

**自製電解質液**：
```
1 公升乾淨水
+ 6 茶匙糖
+ 半茶匙鹽
攪拌均勻後飲用
```

---

### ❓ 不確定食材是否安全

**判斷原則**：
1. 看：顏色、質地是否正常
2. 聞：有無異味、酸味、腐味
3. 摸：有無黏液、軟爛

**決策流程**：
- 任一項異常 → **丟棄**
- 無法確定 → **丟棄**
- 只是過「保存期限」但狀態正常 → 可謹慎使用（煮熟）

> **寧可餓一餐，不可中毒三天**

## ❓ FAQ

**Q1：沒有冰磚怎麼保冷？**
> 可用冷凍的寶特瓶水替代。選購時購買常溫可保存的食材（罐頭、調理包）減少冷藏需求。

**Q2：山上的溪水看起來很乾淨，真的不能直接喝嗎？**
> 不行。即使水質清澈，仍可能含有梨形鞭毛蟲、大腸桿菌等病菌，必須經過淨化。

**Q3：淨水錠會不會影響健康？**
> 正確使用不影響健康。可能有輕微氯味，放置一段時間可揮發掉。

**Q4：腹瀉時可以吃止瀉藥嗎？**
> 若是細菌感染造成的腹瀉，止瀉藥可能使毒素滯留體內。輕微腹瀉以補水為主，嚴重腹瀉應就醫。

**Q5：罐頭過期還能吃嗎？**
> 若外觀正常（無膨脹、無生鏽、開罐無異味），過期短時間內可謹慎食用。有任何異常立即丟棄。

**Q6：用營火煮水要煮多久？**
> 水滾後維持沸騰至少 3 分鐘。海拔越高沸點越低，3000m 以上建議延長至 5 分鐘。

**Q7：生菜沙拉可以帶去露營嗎？**
> 非常不建議。生食蔬菜若未徹底清洗或保存不當，腸胃問題風險高。野外優先選擇熟食。

**Q8：只用清水洗砧板夠乾淨嗎？**
> 不夠。處理生肉後的砧板應用洗潔精清洗，再以酒精噴霧消毒，或使用獨立的生食砧板。
