# 場景化培訓 SOP

> 災難應變系統培訓標準作業程序

---

## 1. 培訓課程概覽

### 1.1 課程清單

| 課程代碼 | 名稱 | 對象 | 時數 |
|----------|------|------|------|
| TR-101 | 系統基礎操作 | 全體人員 | 2h |
| TR-201 | 任務派遣與追蹤 | 隊長/幹部 | 3h |
| TR-301 | 指揮中心操作 | 指揮官 | 4h |
| TR-401 | 系統管理員培訓 | IT 人員 | 6h |

| 系統等級 | 角色 | 必修課程 |

| Level 1 | 志工 | TR-101 |

| Level 4 | 指揮官 | TR-101, TR-201, TR-301 |
| Level 5 | 系統管理員 | TR-401 |

---

## 2. TR-101: 系統基礎操作

### 2.1 學習目標

- 完成帳號登入與個人設定
- 了解系統主要功能區塊
- 能夠接收並回報任務狀態
- 使用即時通訊功能

### 2.2 課程大綱

| 時間 | 主題 | 方式 |
|------|------|------|
| 0:00-0:15 | 系統介紹與登入 | 講授 |
| 0:15-0:45 | 介面導覽 | 實機操作 |
| 0:45-1:15 | 任務接收與回報 | 實機演練 |
| 1:15-1:45 | 即時通訊 | 分組練習 |
| 1:45-2:00 | Q&A | 討論 |

### 2.3 實機演練腳本

```
【情境】颱風期間，您收到一則物資運送任務

1. 開啟 APP → 點選「我的任務」
2. 找到狀態為「待接受」的任務
3. 點選任務 → 閱讀詳情
4. 確認後點選「接受任務」
5. 抵達現場後點選「簽到」
6. 完成後點選「完成任務」並附上照片
```

---

## 3. TR-201: 任務派遣與追蹤

### 3.1 學習目標

- 建立並派遣任務
- 追蹤隊員位置與進度
- 處理任務異常狀況
- 使用團隊管理功能

### 3.2 課程大綱

| 時間 | 主題 | 方式 |
|------|------|------|
| 0:00-0:30 | 任務建立與類型 | 講授 |
| 0:30-1:00 | 人員指派策略 | 案例討論 |
| 1:00-1:30 | 即時追蹤與調度 | 實機操作 |
| 1:30-2:15 | 異常處理演練 | 情境模擬 |
| 2:15-2:45 | 報表與統計 | 實機操作 |
| 2:45-3:00 | 總結 | 討論 |

### 3.3 情境模擬腳本

```
【情境】山區搜救，需派遣 3 隊前往不同區域

1. 進入「指揮中心」→「任務派遣」
2. 點選「新增任務」
3. 設定任務類型：搜索
4. 設定優先等級：高
5. 在地圖上框選搜索區域
6. 選擇指派小隊
7. 確認後點選「派遣」
8. 於「任務監控」頁面追蹤進度
```

---

## 4. TR-301: 指揮中心操作

### 4.1 學習目標

- 啟動與管理任務場次
- 建立指揮鏈架構
- 發布 SITREP 報告
- 生成事後行動報告 (AAR)

### 4.2 課程大綱

| 時間 | 主題 | 方式 |
|------|------|------|
| 0:00-0:45 | 任務場次管理 | 講授+實作 |
| 0:45-1:30 | 指揮鏈配置 | 案例練習 |
| 1:30-2:15 | SITREP 製作 | 實機操作 |
| 2:15-3:00 | 資源調度 | 情境模擬 |
| 3:00-3:45 | AAR 產製 | 實機操作 |
| 3:45-4:00 | 綜合演練 | 分組競賽 |

### 4.3 綜合演練腳本

```
【情境】地震後，EOC 啟動三級應變

1. 建立主任務場次
   - 類型：地震
   - 受影響區域：信義區
   
2. 設定指揮架構
   - 指揮官：[指派]
   - 搜救組：[指派]
   - 醫療組：[指派]
   - 後勤組：[指派]

3. 每 30 分鐘發布 SITREP
   - 人力狀態
   - 資源消耗
   - 傷患統計

4. 任務結束後產製 AAR
```

## 5. 演訓腳本庫

### 5.1 山難搜救演訓 (2小時)

```yaml
scenario: MOUNTAIN_SEARCH_DRILL
duration: 2h
participants: 8-12
roles:
  - 指揮官 x1
  - 搜救隊長 x2
  - 搜救隊員 x6
  - 醫護 x2

phases:
  - name: 報案與任務啟動
    duration: 15m
    actions:
      - 接獲山難報案
      - 建立任務場次
      - 召集人員
      
  - name: 搜索網格劃分
    duration: 20m
    actions:
      - 劃分搜索區域
      - 指派各隊負責區域
      - 確認通訊頻道
      
  - name: 搜索執行
    duration: 45m
    actions:
      - 各隊出發搜索
      - 定時回報位置
      - 模擬發現失蹤者
      
  - name: 檢傷與後送
    duration: 25m
    actions:
      - START 檢傷
      - 安排後送順序
      - 通報醫院
      
  - name: 任務結案
    duration: 15m
    actions:
      - 人員點收
      - 產製 AAR
      - 儲存資料
```

## 6. 評量標準

### 6.1 技能檢核表

| 技能項目 | 合格標準 |
|----------|----------|
| 登入系統 | 30 秒內完成 |
| 接受任務 | 正確識別並接受指定任務 |
| 任務回報 | 含照片與座標 |
| 派遣任務 | 5 分鐘內完成含人員指派 |
| SITREP 製作 | 10 分鐘內完成含必要欄位 |

### 6.2 認證流程

1. 完成線上課程
2. 通過技能測驗 (≥ 80%)
3. 參與實機演練
4. 主管核發認證
