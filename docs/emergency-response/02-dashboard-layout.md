# Emergency Response Dashboard 版面規格

**文件編號**：02  
**版本**：1.0

---

## 1. 佈局結構

採用 **12 欄 Grid 系統**：
- 桌機（≥1200px）：12 欄
- 平板（768-1199px）：2 欄
- 手機（<768px）：1 欄堆疊

---

## 2. 頁面區塊（由上至下）

```

│ [●Online] 最後同步: 14:32:05 │ Emergency Response │ 14:32 🔍 👥3 👤│
```

| 位置 | 內容 |
|------|------|

## 3. RWD 行為

| 斷點 | 行為 |
|------|------|
| 手機 <768px | KPI 變橫向滑動；Quick Actions 置頂；卡片單欄堆疊 |
| 平板 768-1199px | 2 欄佈局；Map 與 Resources 並排 |
| 桌機 ≥1200px | 標準 3 欄佈局 |
