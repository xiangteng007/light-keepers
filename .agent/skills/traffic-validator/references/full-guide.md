# Traffic Validator - Full Guide

Complete guide to validating traffic strategy against Traffic Secrets principles.

## Scoring Criteria

### Dream 100 Clarity (0-10, Weight: 2x)

| Score | Description |
|-------|-------------|
| 9-10 | Clear Dream 100 list, actively engaging, know priorities |
| 7-8 | Good list exists, some engagement, clear targets |
| 5-6 | General idea of where audience is, no formal list |
| 3-4 | Vague sense, no specific names or places |
| 0-2 | No idea where audience congregates |

**Dream 100 Categories:**

| Category | Examples |
|----------|----------|
| Influencers | YouTubers, podcasters, thought leaders |
| Communities | Subreddits, Facebook groups, Slack channels |
| Publications | Blogs, newsletters, media sites |
| Events | Conferences, meetups, webinars |
| Platforms | Specific hashtags, topics, channels |

**Questions to Ask:**

- Name 10 influencers your ideal customer follows
- Name 5 communities where they hang out
- Name 3 publications they read
- Name 3 events they attend

### Hook Strategy (0-10, Weight: 1.5x)

| Score | Description |
|-------|-------------|
| 9-10 | Scroll-stopping hooks, tested and optimized, audience-specific |
| 7-8 | Good hooks, some testing, mostly working |
| 5-6 | Hooks exist but generic or weak |
| 3-4 | Minimal hook strategy, hoping content finds audience |
| 0-2 | No hooks, just posting content |

**Hook Types:**

| Type | Example | When to Use |
|------|---------|-------------|
| Curiosity | "I tried X and this happened..." | Unknown topic |
| Result | "How I got X in Y days" | Proven results |
| Controversy | "Why X is wrong about Y" | Crowded space |
| Story | "The day everything changed..." | Personal brand |
| List | "7 ways to..." | Educational content |

### Work In vs. Buy In (0-10, Weight: 1.5x)

| Score | Description |
|-------|-------------|
| 9-10 | Clear strategy, appropriate for stage, committed execution |
| 7-8 | Good clarity, mostly appropriate, some commitment |
| 5-6 | Mixed approach, unclear primary |
| 3-4 | Dabbling in both, mastering neither |
| 0-2 | No clear strategy |

**Stage Appropriateness:**

| Stage | Recommended |
|-------|-------------|
| Pre-revenue | Work In (validate offer first) |
| Early revenue | Work In primary, test Buy In |
| Established | Mix based on unit economics |
| Scaling | Buy In with proven funnel |

### Funnel Alignment (0-10, Weight: 1.5x)

| Score | Description |
|-------|-------------|
| 9-10 | Perfect alignment, right traffic to right funnel, congruent messaging |
| 7-8 | Good alignment, mostly matched, minor gaps |
| 5-6 | Some alignment issues, mismatched asks |
| 3-4 | Significant misalignment, wrong funnels for traffic |
| 0-2 | Complete mismatch, selling to cold traffic directly |

**Traffic Temperature:**

| Temperature | They Know | Appropriate Funnel |
|-------------|-----------|-------------------|
| Cold | Nothing | Lead magnet, free content, awareness |
| Warm | Problem + solution | Tripwire, webinar, challenge |
| Hot | Ready to buy | Sales page, checkout, booking |

**Red Flags:**

- Sending cold traffic to sales page
- Long-form sales to brand new audience
- No warming sequence
- Same funnel for all traffic

### Audience Ownership (0-10, Weight: 1.5x)

| Score | Description |
|-------|-------------|
| 9-10 | Strong owned audience, growing list, active engagement |
| 7-8 | Good list, regular growth, some engagement |
| 5-6 | List exists, inconsistent growth or use |
| 3-4 | Minimal list, platform dependent |
| 0-2 | No owned audience, 100% platform dependent |

**Ownership Priorities:**

1. **Email list** - Most important, you own it
2. **SMS list** - High engagement, you own it
3. **Community** - High value, you control access
4. **YouTube subscribers** - Algorithm dependent but you own content
5. **Social followers** - Least valuable, platform owns relationship

### Conversion Path (0-10, Weight: 1x)

| Score | Description |
|-------|-------------|
| 9-10 | Clear path, measured conversion, optimized sequence |
| 7-8 | Good path exists, tracking in place |
| 5-6 | Path exists but not optimized or measured |
| 3-4 | Vague path, no measurement |
| 0-2 | No conversion path, traffic goes nowhere |

**Conversion Path Elements:**

- Traffic source → Landing page
- Landing page → Lead capture
- Lead capture → Nurture sequence
- Nurture sequence → Sales
- Sales → Customer
